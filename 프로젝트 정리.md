프로젝트 전체 정리

1. ResearchQ

AI 기반 리서치 및 데이터 인텔리전스 플랫폼.

핵심 개념

대규모 데이터를 수집·분석하고, 인사이트와 리포트를 자동 생성하는 시스템.

단순 대시보드가 아니라:

* 데이터 파이프라인
* AI 분석
* 시각화
* 리포트 생성
* 의사결정 지원

까지 연결되는 형태.

주요 기술

* Python
* SQL
* BigQuery
* FastAPI
* 데이터 시각화
* 머신러닝
* 클라우드 인프라

특징

* 데이터사이언스 역량을 가장 직접적으로 보여줌
* B2B 분석 플랫폼 형태로 확장 가능
* 기술 포트폴리오 가치 높음

어울리는 포지션

* Data Scientist
* Data Engineer
* Analytics Engineer
* AI Research Engineer

⸻

2. Interview Easy

AI 기반 면접 준비 및 커리어 코칭 서비스.

핵심 개념

사용자의:

* 이력서
* 경험
* JD(Job Description)
* 지원 직무

를 기반으로:

* 예상 질문 생성
* STAR 답변 생성
* 모의면접
* 답변 피드백
* 영어 인터뷰 교정

을 제공하는 서비스.

주요 기술

* LLM
* Prompt Engineering
* Resume Parsing
* Speech/Feedback 시스템
* Web SaaS 구조

특징

* MVP 제작 속도가 빠름
* 실제 수요가 매우 명확함
* SaaS 구독 모델 적용 쉬움

타겟 사용자

* 유학생
* Graduate 지원자
* Big4/Consulting 지원자
* 영어 인터뷰 준비생

어울리는 포지션

* AI Engineer
* Product-focused ML Engineer
* Full-stack AI Developer

⸻

3. 영앤리치 리포트

투자·경제·트렌드 기반 인사이트 리포트 프로젝트.

핵심 개념

경제·시장·소비 데이터를 기반으로:

* 투자 인사이트
* 트렌드 분석
* 자산 관련 리포트
* AI 기반 요약 콘텐츠

를 제작하는 콘텐츠/리서치 플랫폼.

특징

* 브랜딩과 콘텐츠 확장성이 강함
* 뉴스레터/SNS 기반 성장 가능
* 개인 브랜드와 연결하기 좋음

가능한 형태

* AI 투자 리포트
* 주간 시장 요약
* 데이터 기반 경제 콘텐츠
* 소비 트렌드 분석

어울리는 방향

* 콘텐츠 비즈니스
* 퍼스널 브랜딩
* 경제/금융 미디어

⸻

4. Vationo

Transformer 기반 머신러닝·딥러닝 퀀트 분석 플랫폼.

핵심 개념

금융 시계열 데이터를 기반으로:

* AI 예측 모델
* 퀀트 전략
* 시그널 생성
* 리스크 분석
* 백테스트

를 수행하는 AI 기반 퀀트 시스템.

주요 기술

* Transformer 기반 시계열 모델
* Deep Learning
* Quantitative Finance
* Backtesting
* MLOps
* Cloud Infrastructure

활용 가능 모델

* Temporal Fusion Transformer
* Informer
* Autoformer
* PatchTST
* LSTM/GRU

데이터 소스

* 가격 데이터
* 거래량
* 뉴스
* Sentiment
* 경제지표
* 재무 데이터

특징

* 가장 기술 난이도가 높음
* 퀀트/ML/데이터엔지니어링 전부 포함
* 강한 엔지니어링 포트폴리오 역할 가능

중요한 포인트

단순 “가격 예측”이 아니라:

* realistic backtesting
* Sharpe ratio
* drawdown 관리
* regime robustness
* transaction cost 고려

등 실제 금융 시스템 관점이 중요함.

어울리는 포지션

* Quant Researcher
* ML Engineer
* AI Engineer
* Quant Developer
* MLOps Engineer
